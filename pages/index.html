Welcome to the Core

As you're browsing the front-end of Know my city, this little server provides the data it needs to feature up to date data.
Let's see how it works.

What data provides?

Currently, the server is specialized in providing data such as:

  • Points of Interest (POI)It could be a bar, a restaurant, a beach resort, whichever group of locations the user demands while using the filters. A local copy is kept if the server goes down.
  • SignsOld touristic signs often don't receive maintenance, and might feature a single translation or none. With this endpoint it allows tourists to read the sign in a digitalized form, usually by scanning a qr code advertised below the sign's content
  • AttractionsSome touristic attractions may have no description at all, and know my city tries to fill in the gaps, by providing a complete description of it.

Can I play with this instance?

Currently, this server is being used as the backend for a legit instance. We kindly ask you to not play with it, your requests might impact the performance of legit users.
You can download the required files here for your own instance.
